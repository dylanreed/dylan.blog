## STYLE NOTES

### Color Palette Suggestions
- **Day mode**: Warm metallics (bronze, copper, gold), amber status lights, green CRT glow, cream/beige plastics
- **Night mode**: Cool blues, purple accents, red emergency lighting, darker metals, minimal warm tones

### Dithering Patterns
- Use ordered/Bayer dithering for gradients
- Checkerboard patterns for mid-tones
- Horizontal scanline effects for CRT screens

### Reference Games
- Space Quest series (Sierra)
- Commander Keen
- Duke Nukem (original 2D)
- Bio Menace
- Cosmo's Cosmic Adventure
- Jazz Jackrabbit (UI elements)
- Wing Commander (cockpit displays)

### Key Differences from Fantasy Theme
| Element | Fantasy | Space |
|---------|---------|-------|
| Scrolls | Parchment paper | CRT monitors |
| Navigation | Wooden signs | LED panel signs |
| Border tiles | Grass/stone | Metal grating |
| Walker | Diver character | Maintenance robot |
| Background | Medieval village | Space station interior |
| Lighting | Torches/candles | Status LEDs/CRT glow |
| Weather effects | Rain/snow | Meteor shower/ice asteroids |
